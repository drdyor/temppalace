# Local LLM Prompt Template for English Zone Dialogues

## Goal
Generate natural A1-A2 English conversations for memory palace zones.
Each conversation must incorporate the zone's vocabulary words naturally.

## Setup

### 1. Prepare prompts
```bash
python scripts/prepare-llm-prompts.py
# This creates llm-prompts.json with one prompt per zone
```

### 2. Load your local model
Works with any Ollama-compatible local model:
- **Mistral 7B** — fast, good quality
- **Mixtral 8x7B** — best quality for dialogue
- **Minerva** (your local Italian-trained model) — for auditing Italian later
- **Llama 3** — solid alternative

```bash
ollama run mistral
# or
ollama run mixtral
```

### 3. System Prompt (copy-paste this)

```
You are a language learning content writer. Write a natural A1-A2 level English conversation for a memory palace language learning app.

RULES:
- Exactly 10 exchanges (lines of dialogue)
- Two speakers: "You" and "Friend" (or context-appropriate: "Barista", "Doctor", "Waiter")
- Simple present tense, common vocabulary, short sentences (1-2 sentences max)
- The conversation must naturally use ALL vocabulary words listed below
- Tone: friendly, everyday, realistic
- No complex grammar. No conditionals. No phrasal verbs.
- Use contractions (I'm, don't, can't) where natural.

OUTPUT FORMAT — strict JSON, no markdown, no explanation:
{"zoneId":"ZID","zoneName":"ZNAME","level":"a2","exchanges":[{"speaker":"You","en":"..."},{"speaker":"Friend","en":"..."}]}
```

### 4. User Prompt (one per zone)

Copy from `llm-prompts.json` or use this template:

```
Zone: The Fridge (Kitchen)
Topic: Cold food and drinks
Vocabulary words to include: water, milk, eggs, butter, cheese, meat, fish, chicken, yogurt, cream, jam

Write the conversation now. Return only the JSON object.
```

## Example Output

```json
{
  "zoneId": "fridge",
  "zoneName": "The Fridge",
  "level": "a2",
  "exchanges": [
    {"speaker": "You", "en": "I am hungry. What do we have in the fridge?"},
    {"speaker": "Friend", "en": "We have eggs, cheese, and some fresh chicken."},
    {"speaker": "You", "en": "Great! I want to make an omelette. Do we have butter?"},
    {"speaker": "Friend", "en": "Yes, the butter is on the top shelf. Next to the milk."},
    {"speaker": "You", "en": "Is the milk still fresh? I do not want sour milk in my coffee."},
    {"speaker": "Friend", "en": "It is fresh. I bought it this morning. There is also yogurt."},
    {"speaker": "You", "en": "I love yogurt with jam. Do we have any strawberry jam?"},
    {"speaker": "Friend", "en": "Yes, but it is at the back. Behind the meat and fish."},
    {"speaker": "You", "en": "We have a lot of food! I am glad we went shopping yesterday."},
    {"speaker": "Friend", "en": "Me too. Let us cook together. I will wash the vegetables."}
  ]
}
```

## Batch Script (Ollama API)

Use `scripts/batch-generate-ollama.py` to process all zones automatically:

```bash
python scripts/batch-generate-ollama.py --model mistral --output src/data/zone-stories-en-batch.ts
```

This feeds each prompt from `llm-prompts.json` to your local Ollama API and collects the JSON outputs.

## Italian Audit with Local Model

Once English is solid, audit the Italian translations using your local model:

```
System: You are an Italian language teacher. Review the following English-Italian dialogue pair for naturalness, correct register (tu vs Lei), and A1-A2 appropriateness. Flag any issues.

English: ...
Italian: ...
```
