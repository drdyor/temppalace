## GENERATION NOTES

### Model Settings
- **Model:** DreamShaper XL v2.1 or similar
- **CFG Scale:** 7-8
- **Steps:** 30-40
- **Sampler:** DPM++ 2M Karras
- **Resolution:** 1024x768 for zones, 1920x1080 for main rooms

### Post-Processing
After generation, verify each image contains ALL vocabulary items listed. If any item is missing, regenerate with more specific prompt.

### File Naming
- Main room: `{room-id}.jpg`
- Zone interior: `{room-id}-{zone-id}.jpg`
