## GENERAL PRINCIPLES FOR ALL ROOMS

### 1. Every Word Must Be Visible
If the vocabulary includes "coltello" (knife), the image MUST show a knife.

### 2. Logical Organization
- Cold items → Fridge
- Dry goods → Pantry  
- Utensils → Drawer
- Clothes → Wardrobe
- Jewelry → Jewelry box
- Body parts → Mirror/reflection

### 3. Image Prompt Structure
```
[Setting description], containing: [list ALL vocabulary items with Italian names in parentheses], [lighting/style details]
```

### 4. Zone Size
Each zone should have 8-15 vocabulary items maximum for clarity.

### 5. Subroom Interior Images
When clicking a zone, the interior image should show a CLOSE-UP view of that specific area with all vocabulary items clearly visible and labeled.
